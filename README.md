# MijonGamesPortfolio
Games Development Unity / Unreal Engine
This portfolio showcases my game development work created over the last three years while studying Games Computing at the University of Lincoln. Throughout my degree I’ve focused heavily on Unity, building projects that explore gameplay programming, systems design, AI behaviour, UI/UX, and polished player experiences.

I’m currently expanding my skill set by learning Unreal Engine and C++, with a focus on understanding engine architecture, memory management, and modern gameplay programming patterns. Future projects in this repository will include Unreal prototypes as I continue developing my skills across both engines.

This repo serves as a collection of:

Unity projects developed during my degree

Personal prototypes and gameplay experiments

Ongoing Unreal Engine C++ learning projects

Documentation, breakdowns, and development notes

My goal is to build a strong cross‑engine foundation and showcase my progression as a gameplay programmer and technical designer.
